---- Installation ----

1. Get Tex Tools 1.9+

2. Copy the contents of this this Zip into your /Documents/TexTools/ModPacks/ folder.

3. In TexTools, in the top left Select Mods -> Import ModPack, then open the 'Better Bikinis' folder and select 'Better Bikinis - Base Install.ttmp'.

4. (Optional) If you wish to install any of the optional installs, you may now install their .ttmps from the /Options/ Folder.

5. Press the 'Import' Button in the pop up dialog.
